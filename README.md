### 项目简介
#### 爬虫实验
目的：
>自动获取链家，自如的上海租房房源信息，分析租房价格和小区二手房均价，以找到性价比最高的房源

信息内容：
>房源名称，地区，户型，朝向，房价，面积，楼层，小区二手房价等
  
技术点：
>· 爬虫，html分析  
>· MongoDB存储，可拓展存储图像信息  
>· 图像识别解析自如的房价加密信息。分别做了图像分割，数字识别，定为，最终提取租房价格
